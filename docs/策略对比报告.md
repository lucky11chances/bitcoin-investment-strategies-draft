# 比特币投资策略对比报告

## 执行摘要

本报告对比了两种比特币投资策略在不同时期的表现：
- **DCA策略 (Dollar-Cost Averaging)**: 每月定投$1,000，共13个月，总投资$13,000
- **Quant策略 (Quantitative)**: 基于10个技术因子的量化交易策略，通过随机游走优化权重以最大化Sharpe比率

---

## 一、训练集表现 (2010-2020)

### DCA策略
| 指标 | 数值 |
|------|------|
| 投资期间 | 2010-08-01 至 2020-12-30 |
| 总投资额 | $13,000 |
| 最终价值 | **$2,312,629,735.98** |
| Sharpe Ratio | **1.78** |
| Sortino Ratio | 2.43 |
| 最大回撤 | -93.07% |
| 投资月数 | 13个月 |

**特点**: 
- 稳健的长期持有策略
- 通过分批买入降低市场波动风险
- Sortino比率优于Sharpe，说明下行风险控制良好

### Quantitative策略
| 指标 | 数值 |
|------|------|
| 投资期间 | 2010年至2020年 |
| 初始资金 | $12,500 |
| 最终价值 | **$29,883,119,586.03** |
| Sharpe Ratio | **2.21** |
| 累计收益率 | 239,064,856.69% |

**技术因子**:
1. 动量指标 (MOM_20, MOM_60)
2. 均线价差 (MA_50_SPREAD, MA_200_SPREAD)
3. 波动率指标 (VOL_20, ATR_PCT_14, VOL_RATIO_20)
4. 价格位置 (PRICE_POS_60, CLOSE_POS)
5. 减半周期 (POST_HALVING)

**优化权重**: [1.70, -6.83, -1.58, 4.51, -0.69, 3.13, 2.07, 7.83, 3.29, -2.19]

**特点**:
- 通过2000步随机游走优化，专门针对训练集调优
- Sharpe比率(2.21)优于DCA(1.78)
- 最终价值约为DCA的13倍
- ⚠️ 警告：策略设计时有意允许过拟合

### 训练集小结
在2010-2020训练集上，Quant策略在风险调整后收益(Sharpe)和绝对收益上均显著优于DCA策略。

## 二、测试集表现 (2023-2024)

### DCA策略
| 指标 | 数值 |
|------|------|
| 投资期间 | 2023-01-01 至 2024-06-27 |
| 总投资额 | $13,000 |
| 最终价值 | **$31,327.61** |
| Sharpe Ratio | **3.04** |
| Sortino Ratio | 6.87 |
| 最大回撤 | -20.28% |
| 累计收益率 | **+141%** |

**表现分析**:
- Sharpe比率从1.78提升至3.04，表现更优
- 回撤从-93%大幅改善至-20%，风险控制显著提升
- 稳健盈利，策略泛化能力强

### Quantitative策略
| 指标 | 数值 |
|------|------|
| 测试期间 | 2024-03-26 至 2024-06-27 |
| 初始资金 | $12,500 |
| 最终价值 | **$9,310.57** |
| Sharpe Ratio | **-5.14** ❌ |
| 累计收益率 | **-25.52%** ❌ |
| 交易天数 | 94天 |

**失败原因分析**:
1. **严重过拟合**: 训练集Sharpe 2.21 → 测试集Sharpe -5.14
2. **滚动窗口失效**: 252天滚动窗口在94天测试集上数据不足
3. **市场环境变化**: 训练集优化的因子权重无法适应新市场
4. **负收益**: 亏损25.52%，资金缩水至$9,310.57

### 测试集小结
DCA策略在测试集上表现稳健甚至更优，而Quant策略完全失效，出现大幅亏损，验证了过拟合的严重性。

## 三、综合对比与结论

### 性能对比表
| 策略 | 训练集Sharpe | 测试集Sharpe | 训练集收益 | 测试集收益 | 稳定性 |
|------|-------------|-------------|-----------|-----------|--------|
| DCA | 1.78 | **3.04** ✅ | $2.31B | **+141%** ✅ | 优秀 |
| Quant | 2.21 | **-5.14** ❌ | $29.88B | **-25.52%** ❌ | 极差 |

### 关键发现

1. **过拟合陷阱**: 
   - Quant策略虽然训练集表现卓越，但测试集完全失效
   - Sharpe比率从正2.21跌至负5.14，性能逆转超过7个标准差

2. **稳健性对比**:
   - DCA策略在两个时期均保持正Sharpe，显示出良好的泛化能力
   - Quant策略无法适应新的市场环境，策略脆弱

3. **实战价值**:
   - DCA简单易执行，心理负担小，长期收益可观
   - Quant策略虽然理论上可优化，但本案例中的过度优化导致实战失败

### 建议

**对于普通投资者**:
- ✅ 推荐DCA策略：简单、稳健、无需频繁调整
- ❌ 不推荐过度优化的Quant策略：过拟合风险极高

**对于量化交易者**:
- 必须在独立测试集验证策略有效性
- 避免在单一时期过度优化参数
- 考虑使用交叉验证和样本外测试
- 设置合理的复杂度约束，防止过拟合
